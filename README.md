#  Node Task CLI

A simple, lightweight, and customizable **command-line Todo Manager** built using **pure Node.js**.  
Manage your daily tasks directly from the terminal â€” without any external frameworks.

This project is perfect for learning Node fundamentals like:

- File System operations (`fs`)
- CLI argument handling (`process.argv`)
- JSON data persistence
- Building real CLI tools
- Modular functions and CRUD operations

---

## Features

- Add new tasks  
- List all tasks  
- Remove tasks by index  
- JSON-based local storage  
- Easy-to-understand code structure  
- No external dependencies
